# Headquarters

Ruby wrapper for the [headquarters
API](https://github.com/groupbuddies/headquarters)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'headquarters'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install headquarters
